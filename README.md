Lab 1.1. Branching, Merging, and Handling Conflicts

Description
By completing this lab, I will simulate a typical workflow used in collaborative development environments. I will practice creating, working on, and merging branches while resolving conflicts, as you would in a real workplace. 
In my Lab, I decided to focus mine around a sparkling water company.
